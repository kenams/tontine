# TontineApp

Application mobile de tontine numerique.

## Structure

- `mobile` : application React Native avec Expo et TypeScript
- `backend` : API Node.js / Express en TypeScript
- `shared` : types metier partages

## Demarrage

### Mobile

```bash
cd mobile
npm install
npm run start
```

### Backend

```bash
cd backend
npm install
npm run dev
```
